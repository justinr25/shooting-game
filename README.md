# Shooting Game

A simple game built with JavaScript and HTML Canvas where the player has to survive for as long as possible by dodging and shooting enemies.

## Demo  
![Demo of Shooting Game](media/shooting-game-demo.gif)

## Usage  
1. Visit the [GitHub Pages link](https://justinr25.github.io/shooting-game/) or open `index.html` in your browser.  
2. Click **Start Game** on the modal to begin.
3. Use **W, A, S, D** keys or the **Arrow Keys** to move your player around the screen.
4. **Click** anywhere on the screen to shoot projectiles in that direction.
5. Defend yourself from the enemies and shoot them to earn points and increase your score.
